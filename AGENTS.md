**Обязательные правила для этого репозитория**
- Всегда используй правила из `.agent/rules/CODEX.md` (если файл существует). Если нет — из `.agent/rules/GEMINI.md`.
- Для каждого запроса выполняй авто-маршрутизацию по `.agent/skills/intelligent-routing/SKILL.md` и выбирай агент(ов).
- Перед ответом для задач с кодом или дизайном прочитай `.agent/agents/<agent>.md` и навыки из его поля `skills:` (см. правила ниже).
- В начале ответа для задач с кодом или дизайном добавляй строку: `Применяю знания @agent...`

**Правила загрузки навыков**
- Запись без `/` в `skills:` означает файл `.agent/skills/<skill>/SKILL.md`.
- Запись с `/` означает путь `.agent/skills/<path>/SKILL.md`.
- Дополнительные файлы внутри навыка читай только если они нужны для текущей задачи.

**Если навык отсутствует**
- Сообщи о пропаже и используй ближайший релевантный навык, не блокируя выполнение задачи.
