# Changelog

Все значимые изменения в модуле Telegram AI Analysis будут документированы в этом файле.

## [Unreleased] - 2026-03-20

### Добавлено

#### Новые классы
- **AnalysisCache** - кэширование результатов AI-анализа для повторного использования
- **AnalysisMetrics** - сбор и анализ метрик использования AI
- **RateLimiter** - защита от злоупотреблений через rate limiting

#### Документация
- **README.md** - полная документация модуля с примерами
- **examples/telegram/example_usage.php** - примеры интеграции новых возможностей
- **CHANGELOG.md** - история изменений

#### Тесты
- **GeminiMatchAnalyzerTest.php** - unit-тесты для GeminiMatchAnalyzer
- **phpunit.xml** - конфигурация PHPUnit для запуска тестов

#### База данных
- **migrations.sql** - SQL-миграции для новых таблиц:
  - `ai_analysis_cache` - кэш результатов
  - `ai_analysis_metrics` - метрики использования
  - Дополнительные поля в `telegram_users` для rate limiting
  - Дополнительные поля в `ai_analysis_requests` для трекинга

#### Конфигурация
- Расширен `.env.example` новыми параметрами:
  - `AI_TRIAL_BALANCE` - пробный баланс для новых пользователей
  - `AI_CACHE_TTL` - время жизни кэша
  - `AI_RATE_LIMIT_PER_USER` - лимит запросов на пользователя
  - `AI_RATE_LIMIT_WINDOW` - окно для rate limiting
  - `GEMINI_TIMEOUT` - таймаут запросов к Gemini
  - `GEMINI_TEMPERATURE` - температура генерации
  - `GEMINI_MAX_OUTPUT_TOKENS` - максимум токенов в ответе

### Улучшено

#### Производительность
- Кэширование повторных запросов анализа одного матча
- Оптимизированные индексы для быстрых запросов статистики
- Возможность инвалидации кэша при обновлении live-данных

#### Мониторинг
- Детальная статистика по алгоритмам, моделям и пользователям
- Отслеживание времени ответа и типов ошибок
- Распределение ошибок для выявления проблем

#### Безопасность
- Rate limiting для защиты от злоупотреблений
- Отслеживание заблокированных пользователей
- Подготовка к шифрованию API-ключей (TODO)

## [1.0.0] - Текущая версия

### Существующий функционал

- Интеграция с Gemini AI для анализа матчей
- Три алгоритма анализа (вероятностный, фаворит в 1T, индивидуальный тотал)
- Система кредитов с пробным балансом
- Ротация API-ключей и моделей с failover
- Автоматический возврат средств при ошибках
- Синхронизация AI-анализа с выводами сканера
- Сохранение истории запросов в БД
- Отслеживание ошибок по ключам и моделям
