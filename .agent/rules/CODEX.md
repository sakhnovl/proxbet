# CODEX.md - Antigravity Kit (Codex)

## Назначение

Это правило заставляет Codex автоматически использовать агентов и навыки из `.agent/` для каждого запроса.

## Обязательный протокол

- Перед ответом определи тип запроса: вопрос, анализ, код/дизайн, отладка, тесты, деплой.
- Выполни авто-маршрутизацию по `.agent/skills/intelligent-routing/SKILL.md`.
- Если пользователь явно указал `@agent`, используй его.
- Перед ответом прочитай `.agent/agents/<agent>.md`.
- Для каждого навыка из `skills:` прочитай `.agent/skills/<skill>/SKILL.md`.

## Загрузка навыков

- Запись без `/` в `skills:` означает файл `.agent/skills/<skill>/SKILL.md`.
- Запись с `/` означает путь `.agent/skills/<path>/SKILL.md`.
- Дополнительные файлы внутри навыка читай только когда они нужны для текущей задачи.

## Формат ответа

- Для задач с кодом или дизайном начинай ответ строкой: `Применяю знания @agent...`
- Для вопросов без кода/дизайна агент может не применяться.

## Если навык отсутствует

- Сообщи о пропаже и используй ближайший релевантный навык, не блокируя выполнение задачи.
